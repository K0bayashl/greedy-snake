# 游戏测试技能

启动游戏并进行功能测试。

## 执行步骤

1. **检查文件** - 确认 game.js, index.html, style.css 存在
2. **启动游戏** - 运行 `start index.html`
3. **功能验证**
   - 方向键控制蛇移动
   - 吃食物增长和得分
   - 碰撞检测（墙壁、自身）
   - 暂停/继续功能
   - 难度切换
   - 双倍分数食物

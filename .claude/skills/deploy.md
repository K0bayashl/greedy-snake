# 部署准备技能

准备将项目部署到静态网站托管服务。

## 执行步骤

1. **检查文件完整性**
   - 确认所有必需文件存在
   - 检查文件大小和格式

2. **优化检查**
   - CSS 是否可以压缩
   - JavaScript 是否可以精简
   - 图片是否优化

3. **生成部署清单**
   ```
   # 部署文件清单
   - index.html
   - style.css
   - game.js
   - README.md (可选)
   ```

4. **部署建议**
   - 推荐平台：GitHub Pages, Netlify, Vercel
   - 提供具体部署命令

## 输出

```
# 部署准备报告

## 文件检查
[文件列表和状态]

## 优化建议
[优化建议]

## 部署方案
### GitHub Pages
[步骤说明]

### Netlify
[步骤说明]
```
